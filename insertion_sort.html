插入排序法（Insertion Sort）

💡 操作原理

插入排序法像是在整理撲克牌。從第二張牌開始，將當前拿到的牌（Key）與左側已排序的牌逐一比較，找到合適的位置後「插入」，直到所有牌都就位為止。

📊 複雜度評估與特性

  • 最壞情況時間複雜度：O(n²)
  • 平均情況時間複雜度：O(n²)
  • 最佳情況時間複雜度：O(n) (當陣列已接近排序完成時)
  • 空間複雜度：O(1)

演算法特性

穩定性 (Stability)：穩定。在比較過程中，若數值相同，不會跨越進行交換，因此能保持原有相對順序。

適用場景：插入排序在「幾乎已排序」的資料中表現極佳。實務上常作為高階演算法（如快速排序 Quick Sort）在遞迴到小區間時的優化方案。

💻 程式碼實作

for i 從 1 到 n-1：
    key = arr[i]
    j = i - 1
    // 將大於 key 的元素向右移
    while j >= 0 且 arr[j] > key：
        arr[j + 1] = arr[j]
        j = j - 1
    arr[j + 1] = key
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
