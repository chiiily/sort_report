插入排序法（Insertion Sort）教學

操作原理

插入排序法是一種逐步建立已排序子序列的演算法。從第二個元素開始，將當前元素插入到前面已排序序列的正確位置，直到整個數列排序完成。

目前 key (持有中)：無

複雜度評估

  • 最壞情況時間複雜度：O(n²)
  • 平均情況時間複雜度：O(n²)
  • 最佳情況時間複雜度：O(n²) (已排序)
  • 空間複雜度：O(1)

插入排序法適合部分已排序數列，亦常見於小型資料集或教學範例。

插入排序法虛擬碼

輸入：長度為 n 的陣列 arr

for i 從 1 到 n-1：
    key ← arr[i]
    j ← i - 1
    while j >= 0 及 arr[j] > key：
        arr[j+1] ← arr[j]
        j ← j - 1
    arr[j+1] ← key

輸出：已排序的 arr
