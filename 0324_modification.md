# 專案技術報告：互動式排序演算法教學平台升級與重構

## 1. 專案背景與目標
本專案旨在開發一個「互動式排序演算法教學網頁」，協助使用者透過視覺化動畫與詳細的文字解說，直觀地理解各種常見排序法（如氣泡排序、選擇排序等）的操作原理。

初期版本已具備完整的排序邏輯與基本的動畫演算，本次升級的主要目標為：**提升使用者介面（UI）質感、優化學習體驗（UX），並擴充教學內容的深度與廣度**。

## 2. 核心架構與技術堆疊
* **標記語言**：HTML5，採用語意化標籤建立清晰的文件結構。
* **樣式表**：CSS3，全面導入 CSS 變數（Variables）進行主題色管理，並結合 Flexbox 與 Grid 佈局技術。
* **互動邏輯**：Vanilla JavaScript（原生 JS），利用 `setTimeout` 實現非同步動畫效果，並透過 DOM 操作即時更新狀態與教學提示。

## 3. 介面優化與視覺設計 (UI/UX)
本次升級大幅度重構了視覺呈現，使其具備現代化網頁應用程式的標準：

* **導入玻璃擬態 (Glassmorphism)**：
  * 背景採用動態漸層結合彩色裝飾球（使用 `filter: blur()`）。
  * 主容器背景設定為 `rgba(255, 255, 255, 0.65)` 並搭配 `backdrop-filter: blur(15px)`，營造半透明磨砂玻璃的高級質感。
* **導覽頁卡片式網格佈局 (Card Grid Layout)**：
  * 將傳統的垂直超連結列表，升級為包含圖示、中英對照與一句話簡介的互動式卡片。
  * 使用 `display: grid` 與 `auto-fit`，實現流暢的響應式設計（RWD）。
* **動畫視覺聚焦優化**：
  * **移除干擾性特效**：移除了陣列元素在比較時的放大效果，維持長條圖大小一致，讓使用者的視覺焦點能準確落在「顏色變化（比較與交換）」上。
