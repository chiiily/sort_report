選擇排序法（Selection Sort）

💡 操作原理

選擇排序法是一種簡單直觀的排序演算法。每一輪會從未排序的部分中選出最小值（或最大值），然後與當前輪的起始位置交換，重複直到整個數列有序。

📊 複雜度評估與特性

  • 最壞情況時間複雜度：O(n²)
  • 平均情況時間複雜度：O(n²)
  • 最佳情況時間複雜度：O(n²)
  • 空間複雜度：O(1)

演算法特性

穩定性 (Stability)：不穩定。因為在交換最小值與當前元素時，可能會破壞原本數值相同元素的相對順序。

適用場景：選擇排序法的最大特點是「交換次數極少（最多 N-1 次）」。當應用環境對於「寫入記憶體」的操作成本遠高於「讀取」時（例如：Flash 記憶體、EEPROM 等硬體寫入壽命有限的場景），選擇排序會比氣泡排序法更具優勢。

💻 程式碼實作

輸入：長度為 n 的陣列 arr

for i 從 0 到 n-2：
    minIndex = i
    for j 從 i+1 到 n-1：
        如果 arr[j] < arr[minIndex]：
            minIndex = j
            
    如果 minIndex != i：
        交換 arr[i] 和 arr[minIndex]

輸出：已排序的 arr
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        // 假設未排序區間的第一個元素是最小值
        int minIndex = i;
        
        // 在剩餘的未排序區間中尋找更小的值
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        
        // 如果找到更小的值，則將其與未排序區間的第一個元素交換
        if (minIndex != i) {
            int temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }
}
def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        # 假設未排序區間的第一個元素是最小值
        min_index = i
        
        # 在剩餘的未排序區間中尋找更小的值
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
                
        # 如果找到更小的值，則將其與未排序區間的第一個元素交換
        if min_index != i:
            arr[i], arr[min_index] = arr[min_index], arr[i]
            
    return arr
