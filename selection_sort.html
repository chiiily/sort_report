選擇排序法（Selection Sort）教學

操作原理

選擇排序法是一種簡單直觀的排序演算法。每一輪會從未排序的部分中選出最小值（或最大值），然後與當前輪的起始位置交換，重複直到整個數列有序。

複雜度評估

  • 最壞情況時間複雜度：O(n²)
  • 平均情況時間複雜度：O(n²)
  • 最佳情況時間複雜度：O(n²)
  • 空間複雜度：O(1)

選擇排序法雖然簡單易懂，但效率較低，適合用於小型資料集或教學用途。

選擇排序法虛擬碼

輸入：長度為 n 的陣列 arr

for i 從 0 到 n-2：
    minIndex ← i
    for j 從 i+1 到 n-1：
        如果 arr[j] < arr[minIndex]：
            minIndex ← j
    如果 minIndex ≠ i：
        交換 arr[i] 和 arr[minIndex]

輸出：已排序的 arr
