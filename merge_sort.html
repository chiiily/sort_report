合併排序法（Merge Sort）

💡 操作原理

合併排序法是一種典型的分治演算法。先遞迴將陣列拆分成小區間，直到每個子區間只有一個元素，再透過合併過程把子區間依序組合成已排序區間。

目前合併區間：無

📊 複雜度評估與特性

  • 時間複雜度：O(n log n) (最壞、平均、最佳皆同)
  • 空間複雜度：O(n)

演算法特性

穩定性 (Stability)：穩定。相同大小的元素在排序後，保持原有的相對位置不變。

適用場景：合併排序法穩定且表現穩定，時間複雜度恆定為 O(n log n)，適合用於需要穩定排序且對性能有要求的場景。缺點是需要額外的 O(n) 空間。常被應用於外部排序（需要讀取大型文件）。

💻 程式碼實作

function mergeSort(arr, left, right):
    if left < right:
        mid = (left + right) // 2
        mergeSort(arr, left, mid)
        mergeSort(arr, mid + 1, right)
        merge(arr, left, mid, right)

function merge(arr, left, mid, right):
    建立臨時陣列 L 和 R
    i = j = 0，k = left
    當 i < length(L) 且 j < length(R) 時：
        如果 L[i] <= R[j]：
            arr[k] = L[i]；i++
        否則：
            arr[k] = R[j]；j++
        k++
    附加剩餘元素
void merge(int arr[], int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;
    
    int *L = new int[n1];
    int *R = new int[n2];
    
    for (int i = 0; i < n1; i++)
        L[i] = arr[left + i];
    for (int j = 0; j < n2; j++)
        R[j] = arr[mid + 1 + j];
    
    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) {
            arr[k++] = L[i++];
        } else {
            arr[k++] = R[j++];
        }
    }
    
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];
    
    delete[] L;
    delete[] R;
}

void mergeSort(int arr[], int left, int right) {
    if (left < right) {
        int mid = left + (right - left) / 2;
        mergeSort(arr, left, mid);
        mergeSort(arr, mid + 1, right);
        merge(arr, left, mid, right);
    }
}
def merge(arr, left, mid, right):
    # 建立左右臨時陣列
    left_part = arr[left:mid+1]
    right_part = arr[mid+1:right+1]
    
    i = j = 0
    k = left
    
    # 合併兩個已排序的部分
    while i < len(left_part) and j < len(right_part):
        if left_part[i] <= right_part[j]:
            arr[k] = left_part[i]
            i += 1
        else:
            arr[k] = right_part[j]
            j += 1
        k += 1
    
    # 附加剩餘元素
    while i < len(left_part):
        arr[k] = left_part[i]
        i += 1
        k += 1
    
    while j < len(right_part):
        arr[k] = right_part[j]
        j += 1
        k += 1

def merge_sort(arr, left=0, right=None):
    if right is None:
        right = len(arr) - 1
    
    if left < right:
        mid = (left + right) // 2
        merge_sort(arr, left, mid)
        merge_sort(arr, mid + 1, right)
        merge(arr, left, mid, right)
    
    return arr
