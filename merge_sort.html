合併排序法（Merge Sort）教學

操作原理

合併排序法是一種典型的分治演算法。先遞迴將陣列拆分成小區間，直到每個子區間只有一個元素，再透過合併過程把子區間依序組合成已排序區間。

目前合併區間：無

複雜度評估

  • 時間複雜度：O(n log n) (最壞、平均、最佳皆同)
  • 空間複雜度：O(n)

合併排序法穩定且在大型數據上表現穩定，適合需要穩定排序的場景。

合併排序法虛擬碼

function mergeSort(arr, left, right):
    if left < right:
        mid = (left + right) // 2
        mergeSort(arr, left, mid)
        mergeSort(arr, mid + 1, right)
        merge(arr, left, mid, right)

function merge(arr, left, mid, right):
    create two temporary arrays L 和 R
    i = j = 0
    k = left
    while i < length(L) and j < length(R):
        if L[i] <= R[j]:
            arr[k] = L[i]; i++
        else:
            arr[k] = R[j]; j++
        k++
    append remaining elements
