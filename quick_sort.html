快速排序法（Quick Sort）教學

操作原理

快速排序法是一種分治演算法。選擇一個基準元素（pivot），將陣列分成兩部分：小於基準的元素和大於基準的元素，然後遞歸排序兩部分。

目前 partition：無

複雜度評估

  • 最壞情況時間複雜度：O(n²)
  • 平均情況時間複雜度：O(n log n)
  • 最佳情況時間複雜度：O(n log n)
  • 空間複雜度：O(log n)

快速排序法在實務中表現優秀，適合大型資料集，但最壞情況下效率較低。

快速排序法虛擬碼

function quickSort(arr, low, high):
    if low < high:
        pi = partition(arr, low, high)
        quickSort(arr, low, pi - 1)
        quickSort(arr, pi + 1, high)

function partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j from low to high - 1:
        if arr[j] < pivot:
            i++
            swap arr[i] and arr[j]
    swap arr[i+1] and arr[high]
    return i+1
